ConcurrentProject2
==================

A concurrent simulation of a TSA line in scala using akka actors.

Instructions:
-------------

	1. Make sure that you have scala 2.11 installed on your machine
	2. Make sure that you have akka for scala 2.11 in your dependencies
	3. compile and run "Main.scala" in scala

	Alternatively:
	3. run executeMe.bat (Executes the jar)

No command line arguments, but you can change the number of lines or 
passengers in the Main.scala file
